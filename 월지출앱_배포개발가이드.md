# 월지출 앱 — 배포 및 모바일 개발 가이드

> 현재 프로젝트: `monthly-expenses` (React + Vite + Electron)  
> 목표: 노트북 + 안드로이드 모바일에서 사용 가능하도록 배포

---

## 전체 작업 순서

| 단계 | 작업 | 난이도 | 소요시간 |
|------|------|--------|----------|
| 1 | GitHub 업로드 | 쉬움 | 10분 |
| 2 | Vercel 배포 | 쉬움 | 5분 |
| 3 | PWA 추가 | 쉬움 | 30분 |
| 4 | 모바일 전용 화면 개발 | 중간 | 2~4시간 |

## STEP 1 — GitHub 업로드

### 1-1. Git 설치 확인
```bash
git --version
```
없으면 https://git-scm.com 에서 설치

### 1-2. Git 최초 설정 (처음 한 번만)
```bash
git config --global user.name "내이름"
git config --global user.email "내이메일@gmail.com"
```

### 1-3. GitHub 저장소 생성
1. https://github.com 로그인
2. 우측 상단 `+` → **New repository**
3. Repository name: `monthly-expenses`
4. **Create repository** 클릭

### 1-4. 프로젝트 폴더에서 터미널 열기
- 파일 탐색기에서 프로젝트 폴더 열기
- 주소창 클릭 → `cmd` 입력 → 엔터
- 또는 빈 곳 우클릭 → "터미널에서 열기"

### 1-5. Git 명령어 순서대로 입력
```bash
git init
git add .
git commit -m "첫 커밋"
git branch -M main
git remote add origin https://github.com/내아이디/monthly-expenses.git
git push -u origin main
```
> `내아이디` → 본인 GitHub 아이디로 교체  
> push 시 GitHub 로그인 팝업 뜨면 로그인

## STEP 2 — Vercel 배포

1. https://vercel.com 접속 → **GitHub으로 로그인**
2. **Add New Project** 클릭
3. `monthly-expenses` 저장소 선택
4. Framework: **Vite** (자동 감지됨)

**3~4분 후 완료!**  
`https://monthly-expenses-xxx.vercel.app` URL 생성  
→ 노트북/모바일 브라우저에서 바로 접속 가능

## STEP 3 — PWA 추가 (홈화면 설치)

### 추가할 파일 목록
```
프로젝트 폴더/
├── public/
│   ├── manifest.json      ← 새로 추가
│   ├── sw.js              ← 새로 추가
│   ├── icon-192.png       ← 새로 추가 (앱 아이콘)
│   └── icon-512.png       ← 새로 추가 (앱 아이콘)
└── index.html             ← 한 줄 추가
```

### 3-1. `public/manifest.json` 생성
```json
{
  "name": "월지출관리",
  "short_name": "월지출",
  "description": "월별 지출 관리 앱",
  "start_url": "/",
  "display": "standalone",
  "orientation": "portrait",
  "background_color": "#0f1117",
  "theme_color": "#0f1117",
  "icons": [
    {
      "src": "/icon-192.png",
      "sizes": "192x192",
      "type": "image/png"
    },
    {
      "src": "/icon-512.png",
      "sizes": "512x512",
      "type": "image/png"
    }
  ]
}
```

### 3-2. `public/sw.js` 생성
```js
const CACHE_NAME = 'mexp-v1';
const URLS = ['/', '/index.html'];

self.addEventListener('install', e => {
  e.waitUntil(caches.open(CACHE_NAME).then(c => c.addAll(URLS)));
});

self.addEventListener('fetch', e => {
  e.respondWith(
    caches.match(e.request).then(r => r || fetch(e.request))
  );
});
```

### 3-3. `index.html` 수정
`<head>` 안에 아래 두 줄 추가:
```html
<link rel="manifest" href="/manifest.json">
<meta name="theme-color" content="#0f1117">
```

### 3-4. `main.jsx` 수정
Service Worker 등록 코드 추가:
```jsx
if ('serviceWorker' in navigator) {
  window.addEventListener('load', () => {
    navigator.serviceWorker.register('/sw.js');
  });
}
```

### 3-5. 아이콘 이미지 준비
- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)
- 원하는 이미지를 위 사이즈로 저장해서 `public/` 폴더에 넣기

### 3-6. 안드로이드에서 설치
1. 크롬에서 Vercel URL 접속
2. 우측 상단 **⋮ 메뉴** → **"앱 설치"** 또는 **"홈 화면에 추가"**
3. 설치 완료 → 앱 아이콘 생성 🎉

## STEP 4 — 모바일 전용 화면 개발

### 구조 설명
기기 너비가 768px 미만이면 모바일 화면, 이상이면 기존 데스크탑 화면을 보여줘요.

### 4-1. `App.jsx` 최상단에 분기 추가
```jsx
export default function App() {
  const isMobile = window.innerWidth < 768;
  return isMobile ? <MobileApp /> : <DesktopApp />;
}
```
> 기존 App 컴포넌트 전체를 `DesktopApp`으로 이름 변경

### 4-2. `MobileApp.jsx` 새 파일 생성
모바일 화면 구성 (하단 탭바 방식):

```jsx
import { useState } from "react";

const G = {
  bg: "#0f1117", bg2: "#1a1d27", bgc: "#1e2130",
  bd: "rgba(255,255,255,0.08)",
  t1: "#e8eaf0", t2: "#8b90a7", tm: "#5a6075",
  blue: "#4a9eff", green: "#34d399", red: "#f87171",
};

export default function MobileApp() {
  const [tab, setTab] = useState("home");

  return (
    <div style={{ height:"100vh", display:"flex", flexDirection:"column",
      background:G.bg, color:G.t1, fontFamily:"'Noto Sans KR',sans-serif" }}>

      {/* 콘텐츠 영역 */}
      <div style={{ flex:1, overflowY:"auto", padding:16 }}>
        {tab === "home"  && <MobileHome />}
        {tab === "add"   && <MobileAdd />}
        {tab === "list"  && <MobileList />}
        {tab === "stats" && <MobileStats />}
      </div>

      {/* 하단 탭바 */}
      <div style={{ display:"flex", borderTop:`1px solid ${G.bd}`,
        background:G.bg2, flexShrink:0 }}>
        {[
          { key:"home",  icon:"🏠", label:"홈" },
          { key:"add",   icon:"➕", label:"입력" },
          { key:"list",  icon:"📋", label:"목록" },
          { key:"stats", icon:"📊", label:"통계" },
        ].map(t => (
          <button key={t.key}
            onClick={() => setTab(t.key)}
            style={{ flex:1, padding:"10px 0", background:"none", border:"none",
              color: tab===t.key ? G.blue : G.tm, cursor:"pointer",
              display:"flex", flexDirection:"column", alignItems:"center", gap:2,
              fontSize:10, fontFamily:"inherit" }}>
            <span style={{ fontSize:20 }}>{t.icon}</span>
            {t.label}
          </button>
        ))}
      </div>
    </div>
  );
}
```

### 4-3. 모바일 화면 4개 구현

**홈 화면** (`MobileHome`) — 이번달 요약
- 수입 / 지출 / 잔액 카드
- 최근 5건 내역

**입력 화면** (`MobileAdd`) — 빠른 지출 추가
- 날짜, 항목명, 금액, 자산 선택
- 저장 버튼

**목록 화면** (`MobileList`) — 이번달 내역
- 날짜순 리스트
- 수입/지출 구분 색상

**통계 화면** (`MobileStats`) — 간단한 차트
- 월별 지출 추이 (LineChart)
- 자산별 비율 (PieChart)

> 데이터는 기존 `localStorage` (`mexp_v1_data`, `mexp_v1_fixed_base`) 그대로 사용

## STEP 5 — (선택) Firebase 데이터 동기화

노트북 ↔ 모바일 간 실시간 데이터 공유가 필요할 때 추가해요.

### 5-1. Firebase 프로젝트 생성
1. https://console.firebase.google.com 접속
2. **프로젝트 추가** → 이름: `monthly-expenses`
3. **Firestore Database** → 시작 → 테스트 모드

### 5-2. Firebase SDK 설치
```bash
npm install firebase
```

### 5-3. `src/firebase.js` 생성
```js
import { initializeApp } from "firebase/app";
import { getFirestore } from "firebase/firestore";

const firebaseConfig = {
  apiKey: "여기에_입력",
  authDomain: "여기에_입력",
  projectId: "여기에_입력",
};

const app = initializeApp(firebaseConfig);
export const db = getFirestore(app);
```

### 5-4. 데이터 저장/불러오기 교체
```js
import { doc, setDoc, getDoc } from "firebase/firestore";
import { db } from "./firebase";

// 저장 (기존 localStorage.setItem 대체)
await setDoc(doc(db, "users", "user1"), { data: JSON.stringify(expenses) });

// 불러오기 (기존 localStorage.getItem 대체)
const snap = await getDoc(doc(db, "users", "user1"));
const data = snap.exists() ? JSON.parse(snap.data().data) : [];
```
